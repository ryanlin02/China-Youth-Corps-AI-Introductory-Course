# 嘉義市婦女學苑 AI 入門應用班成果網站


這個資料夾是課程成果網站的第一版靜態網站專案。

## 目前版本

- 首頁：`index.html`
- 第一堂課內容頁：`courses/c01.html`
- 共用樣式：`assets/styles.css`
- 共用互動：`assets/app.js`
- 首頁示意視覺：`assets/visuals/`

## 使用方式

直接用瀏覽器開啟 `index.html` 即可預覽。

這一版沒有使用 npm、框架或後端服務，因此之後可以整個資料夾上傳到免費靜態網站空間，例如 GitHub Pages、Netlify、Cloudflare Pages。
